# Sunnyside

```bash
Make some scrambled eggs

Usage: sunnyside --target <TARGET> --shift <SHIFT> --key <KEY>

Options:
  -t, --target <TARGET>  File target
  -s, --shift <SHIFT>    Shift amount
  -k, --key <KEY>        Scramble key
  -h, --help             Print help
  -V, --version          Print version
```
